# Looper

![logo.png]

Personal webapp to manage repetitive tasks & spontaneous ideas.

![screenshot.png]

## Why?

- I have a lot of asynchronous tasks and impulsive ideas, I want to store them somewhere
- Conventional task management apps are slow an cranky, I want mine to be as smooth as possible
- Never forget a birthday
- Never forget an idea
- All scheduled repetitive tasks in the same feed
- Want to do something later, but afraid you will forget? Now you won't
- Lose one of the excuses why the house is a mess

## Features

- The workflow is text+tags: you push a bunch of ideas & tasks during the day, then (or even before) you sort them to tags by mentioning a tag in the task's text
- The killer feature is unpublished: I made a shortcut for my iPhone, triple tap on the back allows to write down an idea without even unlocking the phone
- Cron-based expressions for repetitive tasks
